# RNGP Patches

This repository contains patches and modifications for the EverQuest emulator RNGP (RNGP Patcher).

## Project Structure

The repository includes various game assets and data files:

- **Maps/** - Map files for different zones
- **Resources/** - Resource files including character data and sound configurations
- **SpellEffects/** - Visual effects for spells
- **uifiles/** - User interface files including textures and load screens

## Files Included

The repository contains numerous .s3d, .txt, .cfg, .mp3, .tga, and other game-related files that make up the EQ emulator patch set.

## Usage

This repository is intended for use with the RNGP emulator project. Please refer to the main project documentation for usage instructions.
